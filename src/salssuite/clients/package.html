Utility classes used by all clients. The actual clients can be found in their respective packages under salssuite.client.*

The {@link salssuite.clients.ConnectDialog} is a dialog that clients can use to try and connect to a user-specified server. All clients use it to let the user determine the IP address and port of the server to connect to as well as the username and password the user wants to authenticate with.

The {@link salssuite.clients.NameToIDClient} is (as the name suggests) rather itself a client, but as it will usually not be used alone it is not located in its own client package. It provides the user with the possibility to 'convert' a given citizen's ID to the citizen's name or vice versa.

Generic components and utility programmes.

The generic component {@link salssuite.util.gui.FilterPanel} can be used to filter almost arbitrary data from a database according to user input. It allows for very straightforward implementations of filtering and should be suited for many purposes.

The generic component {@link salssuite.util.gui.PasswordPanel} can be used by modules to automatise authentication with the server.

The {@link salssuite.util.gui.ExceptionDisplayDialog} can be used to display Java Exceptions to the user.

The {@link salssuite.util.gui.ProgressDialog} is a tiny helper dialog to display especially indeterminate progress along with a message.

{@link salssuite.util.gui.Installer} and {@link salssuite.util.gui.Uninstaller} are used for convenient deployment of the SalSSuite. They are rather primitive, but easy to use.

The {@link salssuite.util.gui.DBTransfer} is a special tool for transferring database tables from one database to another.

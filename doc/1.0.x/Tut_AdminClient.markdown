# AdminClient/Server-Oberfl&auml;che

Der AdminClient (`AdminClient.jar`) und die graphische Oberfl&auml;che des Servers bieten fast genau die gleichen Funktionen, deswegen werden sie hier gemeinsam behandelt.

## AdminClient
Sobald Sie sich angemeldet haben, erscheint folgendes Fenster:

![AdminClient Login-Screen](http://img39.imageshack.us/img39/892/adminclientloginscreen.png)

Wie Sie sehen, haben Sie am oberen Fensterrand die M&ouml;glichkeit, zwischen f&uuml;nf Tabs zu wechseln. Diese werden der &Uuml;bersicht halber einzeln vorgestellt:

* [B&uuml;rgerverwaltung](Tut_CitizenManagement.html)  
* [Betriebe](Tut_CompanyManagement.html)  
* [Zoll](Tut_Duty.html)  
* [Benutzer](Tut_Accountmanagement.html)   
* [Logout](Tut_Logout.html)

## Server-Oberfl&auml;che
Wenn Sie [den Server mit graphischer Oberfl&auml;che starten](Tut_ServerStart_GUI.html), sehen Sie oben vier der genannten f&uuml;nf Tabs und statt dem "Benutzer"-Tab einen zus&auml;tzlichen:

* [Server](Tut_ServerPanel.html)   

## Berechtigungen
Um den AdminClient zu benutzen, brauchen Sie zwei Berechtigungen:

1. f&uuml;r das Modul "Administration"  
2. f&uuml;r das Modul, auf das Sie zugreifen wollen, also z.B. "Betriebe"  

Um die graphische Oberfl&auml;che des Servers zu benutzen gen&uuml;gt letztere.
